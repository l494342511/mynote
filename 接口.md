# 接口

## 1. 参数控制注解@RequestParam

用于注解接口的参数，该注解有三个参数

| 参数     | 默认值 | 作用                                                     |
| -------- | ------ | -------------------------------------------------------- |
| value    | ""     | 指定该参数的默认值                                       |
| name     | ""     | 指定该参数的在访问路径中的参数名                         |
| required | true   | 当为true时，必须给该参数传递一个非空值，或指定非空默认值 |
